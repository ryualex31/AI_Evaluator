## ⚙️ Run Locally

```bash
git clone <repo-link>
cd <repo>

pip install -r requirements.txt
streamlit run app.py
```

Create `.env`:

```
AZURE_OPENAI_API_KEY=your_key
AZURE_OPENAI_ENDPOINT=your_endpoint
AZURE_OPENAI_DEPLOYMENT=gpt-4o
AZURE_OPENAI_API_VERSION=2024-02-15-preview
```
